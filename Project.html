  • Database Filtering Prgram

    Designed a custom application in the Python programming language that filters a large data set and inserts the filtered data into an SQLite database. The target dataset is a collection of 1,000 data entries. Each entry has information about a single meteorite landing on Earth and consists of the following twelve data fields: Name, Id, name type, reclass, mass (g), fall, year, reclat, reclong, Geolocation, States, Countries. The data used in this program is gathered from a NASA data repository using HTTP GET requests. This program builds a SQLite database with seven tables, with each table corresponding to a continental region on Earth. The purpose of this program is to determine which meteorites fell in regions on earth based on the geolocation given for each meteorite in the NASA data. (With each of the continental regions having a bounding box associated with it. Ie North, America, South America, Africa …etc.
